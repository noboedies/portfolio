Overview

DivineFlow is a responsive front-end web portal designed to handle crowd management challenges at large pilgrimage sites. It provides real-time awareness of visitor density, slot-based bookings, queue estimates, and emergency alerts. The project focuses on creating a seamless, user-friendly interface that supports visitors, administrators, and volunteers in maintaining safety and efficiency.

Key Features

  • Live crowd-density and queue-time dashboard for major temples.
  • Slot-based booking interface for pilgrims and devotees.
  • Emergency alert section for real-time safety notifications.
  • Volunteer registration and management section.
  • Multi-site dashboard (e.g., Somnath, Dwarka, Varanasi).
  • Completely responsive design with mobile compatibility.

Technologies Used

  • Frontend: HTML5, CSS3, JavaScript (Vanilla)
  • UI Design: Custom responsive layout with cards and dashboards
  • Hosting: GitHub Pages
  • Version Control: Git
  • Assets: Icons, images of temples, and alert visuals

My Role & Responsibilities

  • Designed and developed the entire front-end architecture and UI layout.
  • Created dynamic components for crowd data, queue time, and alerts.
  • Implemented responsive design for multi-device support.
  • Deployed project using GitHub Pages for public access.
  • Ensured clarity, usability, and clean interface for all sections.

Learning Outcomes

  • Enhanced skills in front-end development and responsive UI design.
  • Gained practical experience in deploying web projects on GitHub Pages.
  • Developed understanding of data-driven UI for real-world applications.

Future Enhancements

  • Integrate backend API for live data feeds (Node.js / Express).
  • Add authentication for admins and volunteers.
  • Include real-time analytics using Chart.js or D3.js.
  • Implement multilingual support and push notifications for alerts.
  • Convert into a PWA (Progressive Web App) for offline usage.

Why This Project Matters

Managing large crowds at pilgrimage sites is a real-world challenge that impacts safety and comfort for millions of visitors. This project showcases how technology and design can come together to create a meaningful, user-centric solution. It highlights not just technical expertise in HTML, CSS, and JavaScript, but also strong UI/UX thinking and problem-solving ability.
